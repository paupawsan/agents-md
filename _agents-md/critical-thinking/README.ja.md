# クリティカルシンキングドキュメント

このフォルダには、エージェントのクリティカルシンキングおよび検証機能に関するドキュメントが含まれています。

## ファイル

- `README.md` - この概要
- `verification-protocols.md` - グラウンドチェックと検証ワークフロー
- `evidence-based-reasoning.md` - クリティカルシンキングと異議申し立てプロトコル
- `hallucination-prevention.md` - 誤情報を避けるためのガイドライン

## エージェントの動作

クリティカルシンキングはメモリ管理とは別のエージェント動作で、以下のことに焦点を当てています：
- 事実検証とグラウンドチェック
- 証拠に基づく推論
- 幻覚の回避
- ユーザーとの敬意ある異議申し立て

完全なクリティカルシンキングガイドラインについては `AGENTS.md` を参照してください。
