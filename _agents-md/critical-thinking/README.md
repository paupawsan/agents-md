# Critical Thinking Documentation

This folder contains documentation for the agent's critical thinking and verification capabilities.

## Files

- `README.md` - This overview
- `verification-protocols.md` - Ground checking and verification workflows
- `evidence-based-reasoning.md` - Critical thinking and disagreement protocols
- `hallucination-prevention.md` - Guidelines for avoiding false information

## Agent Behaviors

Critical thinking is a separate agent behavior from memory management, focused on:
- Fact verification and ground checking
- Evidence-based reasoning
- Avoiding hallucination
- Respectful disagreement with users

See `AGENTS.md` for the complete critical thinking guidelines.
